🍄 Metrognome 🍄

Where Every Beat is Garden Fresh!

A whimsical terminal-based metronome featuring garden gnomes, twinkling stars, and delightful animations. Built with Go and Bubble Tea for the ultimate CLI experience.

Get Started

Magical Features

Everything a garden gnome needs for perfect timing

🎵

Multiple Time Signatures

From simple 4/4 garden marches to complex 7/8 gnome riddles, support for all your rhythmic needs.

✨

Twinkling Star Animation

ASCII stars dance across your terminal in perfect sync with the beat, creating a magical atmosphere.

🎭

Animated Garden Gnomes

Watch gnomes light up with each beat, featuring alternating poses and special colors for downbeats.

🎨

Pulsing Visual Feedback

Color-coded borders pulse with distinct hues for each beat animation frame.

⚙️

Garden-Themed Presets

Pre-configured rhythms from "Peaceful Garden Stroll" to "Pixie Dust Presto" for every occasion.

🔊

Cross-Platform Sound

Different audio cues for downbeats and regular beats on macOS, Linux, and Windows.

Installation

Get your garden gnome metronome up and running

# Clone the repository git clone https://github.com/yourusername/metrognome.git cd metrognome # Install dependencies go mod tidy # Run the application go run . # Or build and install go build -o metrognome ./metrognome

Quick Commands

Space

Start/Stop the metronome

↑/↓

Adjust BPM (±5)

←/→

Change time signature

P

Browse garden presets

S

Toggle sound on/off

?

Show help guide
